How to translate the Geo POS

Arabic, Bengali, Chinese-simplified, Chinese-traditional, Czech, Dutch, French, German, Greek, Hindi, Indonesian, Italian, Japanese, Korean, Latin, Polish, Portuguese, Russian, Spanish, Swedish, Turkish, Urdu. Application is fully translated in above languages and files included.

As translation is the subject of improvement so you may find some inappropriate words in translation. It will be great if you help us to improve translation. You can contribute the language files improvements, proper credits will be given in the application documentation. You can translate it to your language if your language is not listed above.

Note! You can edit your language file in your_project/application/languages/



Tips

Please create a backup of original language file ,it is mandatory. Additionally, it will be helpful in case of translation failure or raise an issue. Please follow these steps.

  1. Go to your_project/application/languages/
  2. Open the file with notepad or any code editor like notepad++
  3. Do not open the file with Microsoft word like editors, they are not designed for coding.
  4. Do not use double quote (") between words and be careful about special characters.

Example: 


$lang['Date'] = "Datum";
$lang['Edit'] = "Bearbeiten";
